Deleting an Admin

Description

This section will guide the admin through deleting an existing Admin on the Louman Website.

Prerequisites

Admin must be logged into the system

Admin must exist

Deleting an Admin process

Step 1: Select the Admin details option on the side bar.

Step 2: Select the delete button in the admin detail table on the admin detail screen.

Step 3: Click on the yes button on the confirmation notification to confirm the deleting or the cancel button to return to the Admin detail screen.

Exercise

Click the delete button in the Admin detail table on the Admin detail screen.

Click the yes button to delete the Admin.

Click the Cancel button to return to Admin detail screen.

Summary

You have now learnt how to delete an Admin successfully.
