Updating of an Admin

Description

This section will guide the admin through updating an existing Admins detail on the Louman Website.

Prerequisites

Admin must be logged into the system

Admin must exist on the system

Updating Admin details process

Step 1: Select the Admin details option on the side bar.

Step 2: Select the view button in the admin detail table on the admin detail screen.

Step 3: Change the relevant details on the view Admin detail screen.

Step 4 : Click on the Update button to proceed with the admin update or the Cancel button to stop the process.

Step 5: Click on the button on the confirmation notification to confirm the update or the cancel button to return to the view Admin detail screen.
