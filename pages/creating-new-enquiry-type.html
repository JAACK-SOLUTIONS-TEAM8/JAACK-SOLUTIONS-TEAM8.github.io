Creating new enquiry type 

Description

This section will guide the admin through adding a new enquiry type on the Louman Website.

Prerequisites

Admin must be logged into the system

Adding a new enquiry type process

Step 1: Select the enquiry type option on the side bar.

Step 2: Select the add button on the enquiry type details screen.

Step 3: Enter information in all the empty text fields, all fields need to be populated with the correct information relating to the enquiry type item being added.

Step 4 : Click on the Save button to proceed with the enquiry type addition or the Cancel button to stop the process.

Step 5: Click on the yes button on the confirmation notification to confirm the addition or the cancel button to return to the enquiry type details screen.
