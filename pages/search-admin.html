Search Admin

Description

This section will guide the admin through searching an Admin on the Louman Website.

Prerequisites

Admin must be logged into the system

Admin must exist on the system

Searching of an Admin process

Step 1: Select the Admin details option on the side bar.

Step 2: Click on the Search bar on the admin detail screen.

Step 3: Input the search criteria.

Step 4: Click on the search button to proceed with the search.
