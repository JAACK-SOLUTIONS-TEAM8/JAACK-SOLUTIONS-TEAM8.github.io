Search location

Description

This section will guide the admin through searching a location on the Louman Website.

Prerequisites

•        Admin must be logged into the system

•        Location must exist on the system

Searching of a location process

Step 1: Select the Location details option on the side bar.

Step 2: Click on the Search bar on the Location detail screen.

Step 3: Input the search criteria.

Step 4: Click on the search button to proceed with the search.

Exercise

•        Click on the Search bar on the Location detail screen.

•        Type in the search criteria based on the location area.
