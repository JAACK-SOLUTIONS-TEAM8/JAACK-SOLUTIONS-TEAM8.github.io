Deleting an employee

Description

This section will guide the admin through deleting an existing employee on the Louman Website.

Prerequisites

•        Admin must be logged into the system

•        Employee must exist

Deleting an employee process

Step 1: Select the Employee details option on the side bar.

Step 2: Select the delete button in the Employee detail table on the Employee detail screen.

Step 3: Click on the yes button on the confirmation notification to confirm the deleting or the cancel button to return to the employee detail screen.

Exercise

•        Click the delete button in the Employee detail table on the Employee detail screen.

•        Click the yes button to delete the employee.

•        Click the Cancel button to return to Employee detail screen.

Summary

You have now learnt how to delete an employee successfully.
