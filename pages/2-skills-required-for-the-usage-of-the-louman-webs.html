2. Skills required for the usage of the Louman Website and mobile application

The following skills are required for a user to use the Louman System:

1.      Basic computer skills such as:

a.      The ability to use and understand a computer’s functions.

b.      The ability to access and use the computer’s web browser

2.      The user must have a basic understanding of the functionality of the Louman mobile app and website. A User manual as well as a training manual is available to help the user further understand how to use the system.Â 

3.      The user must be understand how to download a mobile application.

4.      To access the Louman System ,the following is needed:

a.      An internet connection

b.      A compatible desktop browser such as Google Chrome
