Timer configuration

Description

This section will guide the admin through configuring a timer on the Louman Website.

Prerequisites

Admin must be logged into the system

Timer configuration process

Step 1: Select the manage timer option on the side bar.

Step 2: Enter information in all the empty text fields, all fields need to be populated with the correct information relating to the timer.

 

Step 3 : Click on the update button to proceed with the timer update or the Cancel button to stop the process.
