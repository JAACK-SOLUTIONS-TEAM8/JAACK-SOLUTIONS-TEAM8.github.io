Adding of a delivery type

Description

This section will guide the admin through adding a new delivery type on the Louman Website.

Prerequisites

Admin must be logged into the system

Adding a new delivery type process

Step 1: Select the delivery type option on the side bar.

Step 2: Select the add button on the delivery type details screen.

Step 3: Enter information in all the empty text fields, all fields need to be populated with the correct information relating to the delivery type being added.

Step 4 : Click on the Save button to proceed with the delivery type addition or the Cancel button to stop the process.

Step 5: Click on the yes button on the confirmation notification to confirm the addition or the cancel button to return to the delivery type screen.
