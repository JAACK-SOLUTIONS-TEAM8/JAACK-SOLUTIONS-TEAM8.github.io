Updating of a product size

Description

This section will guide the admin through updating an existing product size details on the Louman Website.

Prerequisites

•        Admin must be logged into the system

•        Product size must exist on the system

Updating product size details process

Step 1: Select the Add product size option on the side bar.

Step 2: Select the view button in the product size table on the product size details screen.

Step 3: Change the relevant details on the add product size screen.

 

Step 4 : Click on the Update button to proceed with the update or the Cancel button to stop the process.

Step 5: Click on the yes button on the confirmation notification to confirm the update or the cancel button to return to the add product size screen.
