Updating of a delivery type

Description

This section will guide the admin through updating an existing delivery type details on the Louman Website.

Prerequisites

•        Admin must be logged into the system

•        Delivery type must exist on the system

Updating delivery type details process

Step 1: Select the delivery type option on the side bar.

Step 2: Select the view button in the delivery type table on the delivery type details screen.

Step 3: Change the relevant details on the delivery type screen.

 

Step 4 : Click on the Update button to proceed with the delivery type update or the Cancel button to stop the process.

Step 5: Click on the yes button on the confirmation notification to confirm the update or the cancel button to return to the view delivery type screen.

Exercise

•        Click the view button in the delivery type table on the delivery type details screen.

•        Update the delivery type with the following details:

Field

Test Data

Name 

Pick up

 

•        Click the update button to update the delivery type.

•         Click the Cancel button to return to delivery type screen.

Summary

You have now learnt how to update delivery type details successfully.

In order to successfully update a delivery type all, the required fields need to be provided.
