Updating of an enquiry type

Description

This section will guide the admin through updating an existing enquiry types details on the Louman Website.

Prerequisites

•        Admin must be logged into the system

•        Enquiry type must exist on the system

Updating team details process

Step 1: Select the enquiry type option on the side bar.

Step 2: Select the view button in the enquiries table on the enquiries screen.

Step 3: Change the relevant details on the view enquiry screen.

Step 4 : Click on the Update button to proceed with the enquiry type update or the Cancel button to stop the process.

Step 5: Click on the yes button on the confirmation notification to confirm the update or the cancel button to return to the enquiries screen.
