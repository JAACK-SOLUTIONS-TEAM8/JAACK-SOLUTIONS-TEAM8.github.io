Generate attendance report

Description

This section will guide the admin through generating attendance report on the Louman Website.

Prerequisites

•        Admin must be logged into the system

Generate attendance report process

Step 1 Select the attendance report option on the side bar.

Step 2: Select a date from the calendar search bar.

 

Step 3: Click on the generate report button in the team detail table for the specific team .

Step 4 : Click on the report download displayed at the bottom of the screen. The attendance report pdf will display.

Exercise

•        Click the calendar search bar and select a date to generate a report for.

•        Click the generate report button

•        Open the downloaded report on the bottom left of the screen to open the pdf.

Summary

You have now learnt how to download an attendance report successfully.
