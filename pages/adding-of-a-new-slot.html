Adding of a new slot

Description

This section will guide the admin through adding a new meeting slot on the Louman Website.

Prerequisites

Admin must be logged into the system

Adding a new slot process

Step 1: Select the slot details option on the side bar.

Step 2: Select the add slot button on the meeting slot details screen.

Step 3: Enter information in all the empty text fields, all fields need to be populated with the correct information relating to the slot being added.

Step 4 : Click on the Save button to proceed with the slot addition or the Cancel button to stop the process.

Step 5: Click on the yes button on the confirmation notification to confirm the addition or the cancel button to return to the add slot screen.

Exercise

•        Click the Add slot button (top left) on the meeting slot details screen.

•        Submit new slot details with the following details:

•        Click the save button to add the slot.

•         Click the Cancel button to return to add slot screen.
